  3. Outline next step here like waiting for a code

    Once you complete the first step of your registration, a CRA security code will be sent to you by mail. The CRA security code has an expiry date, which will be indicated in the letter you receive.

    Before the expiry date, sign in with the option you registered with and follow the online instructions until you are prompted for the security code.

    Get a new security code

    If your security code expires, you will have to get a new security code issued to you.

    Need immediate access? Get limited access without a security code

    You can get limited service immediately, similar to a Guest account by beginning the registration process and proving your identity. You do not need to finish the full registration process (mail or call for a security code, set up security questions etc.) to access the following:

    • View status of your tax return
    • Notification preferences
    • Request a remittance voucher
    • View notice of assessment or reassessment (simple view, lines X, Y and Z)
    • View RRSP deduction limit and TFSA contribution room
    • Pay by pre-authorized debit (create new agreement)
